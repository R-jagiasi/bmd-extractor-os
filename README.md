# BMD Extractor

A tool for extracting Bone Mineral Density data from medical PDF reports.

---

## This project has moved

**Check out the live app:** [bmd-extractor.vercel.app](https://bmd-extractor.vercel.app)

---

### What it does

- Extracts BMD values, T-scores, and Z-scores from DEXA scan PDFs
- Automatically classifies bone health using WHO criteria
- Generates professional Word documents for clinical use

### Tech Stack

- Python / Flask
- Regex-based PDF extraction (no external APIs)
- SQLite for history
- Vanilla HTML/CSS/JS frontend
